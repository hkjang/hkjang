## About

I build enterprise-grade **AI, data, and platform products** that transform complex operational challenges into reliable and scalable software.

My work focuses on the intersection of:

* Enterprise AI platforms and local LLM infrastructure
* AI gateways, model routing, agents, and developer experience
* Data products, Text2SQL, metadata, and analytics platforms
* Kubernetes operations, DevSecOps, observability, and FinOps
* FinTech workflows and business process automation

I enjoy building practical systems that connect **technology, operations, governance, and business value**.

## Current Focus

| Area                 | What I am building                                                                        |
| -------------------- | ----------------------------------------------------------------------------------------- |
| Enterprise AI        | Local LLM platforms, AI gateways, model routers, agents, and secure AI operations         |
| Data Intelligence    | Text2SQL, metadata automation, data products, analytics, and AI-assisted data workbenches |
| Platform Engineering | Kubernetes operations, developer portals, GitOps, DevSecOps, and observability            |
| FinTech              | Financial data products, workflow automation, and intelligent decision-support systems    |
| Open Source          | Production-oriented tools designed for enterprise and offline environments                |
